# Basic Gulpfile

>    * Vanilla JavaScript ES6
>    * Sass Preprocessor
>    * Pug template engine
>    * Gulp as task runner
---

###  Required
* nodejs

>
>    visite [Nodejs](http://nodejs.org/) site to install
>

* Gulp in global

```
$ npm install gulp-cli -g

```


### Installation


```
$ cd 'to the folder cloned' 

$ npm i 
or
$ yarn

```



### Developpement Serve

```
$ gulp

```


### Serve the Fully Built & Optimized Site

```
$ gulp build

```
